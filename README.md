Este proyecto implementa un modelo de regresión lineal simple utilizando TensorFlow/Keras para predecir valores de salida y a partir de entradas x.
El objetivo es entrenar una neurona que aprenda la relación lineal entre los datos y pueda realizar predicciones para nuevos valores.

Tecnologías utilizadas
-Python 3.x
-NumPy
-TensorFlow / Keras
-Matplotlib (para visualización)
